django-autoload
===============

Copy of <https://bitbucket.org/twanschik/django-autoload>

This Python module is part of the django-nonrel suite
by Waldemar Kornewald, Thomas Wanschik and other 
contributors.

It is ready to be imported into a Django project as 
subdirectory `autoload`.  
See <https://github.com/hflicka/django-testapp> as
an example of how to import submodules into a git 
repository.
